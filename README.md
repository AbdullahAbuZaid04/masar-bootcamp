# Masar Bootcamp


## نظرة عامة
هذا المستودع يحتوي على تطبيق **Masar Bootcamp** المبني بـ **Next.js 16 (App Router)**، React، ومكوّنات Tailwind ذات التصميم الفاخر. يتيح للمستخدمين:
- عرض قائمة بنقاط الإغاثة النشطة مع الحالة، المؤسسة، الموقع، والنوع.
- إضافة نقطة إغاثة جديدة عبر نموذج بسيط (`FormSection`).
- حذف نقاط مباشرة من الواجهة.


## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** JavaScript (React Server Components)
- **Styling:** Tailwind CSS
- **Icons:** lucide‑react
- **Data API:** `https://aid.masartech.org/api/points`


## Getting Started
### Clone the repo
```bash
git clone https://github.com/AbdullahAbuZaid04/masar-bootcamp.git
```
### Install dependencies
```bash
npm install
```
### Run the development server
```bash
npm run dev
```
